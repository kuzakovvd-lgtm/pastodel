# Pastodel

Статический многостраничный сайт бренда Pastodel на Astro. Проект ориентирован на SEO, адаптивную верстку, чистую структуру кода и простой деплой на Ubuntu-сервер с Nginx.

## Что внутри

- Astro со статической сборкой
- Главная, каталог, страницы продуктов, партнерский раздел, где купить и контакты
- SEO-мета для всех страниц
- `Organization` schema на главной
- `Product` schema на страницах продуктов
- `robots.txt` и sitemap через `@astrojs/sitemap`
- Адаптивная верстка с приоритетом мобильных устройств
- Минимальный клиентский JavaScript только для фильтра каталога

## Локальный запуск

Требования:

- Node.js 20+
- npm 10+

Установка и запуск:

```bash
npm install
npm run dev
```

По умолчанию Astro поднимет локальный сервер. Для разработки с доступом из сети можно использовать:

```bash
npm run dev -- --host
```

## Сборка

```bash
npm run build
```

Готовая статическая версия будет собрана в папку `dist/`.

Локальный предпросмотр собранной версии:

```bash
npm run preview
```

## Деплой на Ubuntu + Nginx

### Вариант 1. Собирать локально и выкладывать только статику

1. Выполните сборку:

```bash
npm install
npm run build
```

2. Создайте директорию на сервере:

```bash
sudo mkdir -p /var/www/pastodel
```

3. Загрузите содержимое `dist/` на сервер, например через `rsync`:

```bash
rsync -av --delete dist/ user@server:/var/www/pastodel/dist/
```

4. Подключите Nginx-конфиг из файла [`DEPLOY_NGINX.md`](./DEPLOY_NGINX.md).

5. Проверьте конфиг и перезагрузите Nginx:

```bash
sudo nginx -t
sudo systemctl reload nginx
```

### Вариант 2. Собирать на сервере

1. Установите Node.js 20+ и npm.
2. Скопируйте весь проект на сервер.
3. Выполните:

```bash
npm install
npm run build
```

4. Укажите `root` в Nginx на `/var/www/pastodel/dist`.

## Формы

Формы на страницах `Партнерам` и `Контакты` пока работают как статические заглушки. Для рабочей версии можно подключить:

- собственный серверный обработчик
- серверную функцию
- CRM или webhook
- внешний сервис обработки форм

Достаточно заменить `action="#"` на нужный URL и, при необходимости, добавить обработку HTTP POST.
